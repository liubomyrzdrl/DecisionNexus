# DecisionNexus

AI-powered Decision Intelligence Platform for incident analysis, anomaly detection, and smarter operational decisions.

---

## 🧠 About

DecisionNexus helps teams:

- Detect anomalies and unusual events
- Analyze incidents in a structured way
- Learn from past cases
- Get AI-powered recommendations

The platform is designed for:
- Operations teams
- Engineers
- Analysts

---

---

### Frontend
- Next.js (App Router)
- TypeScript
- Tailwind CSS

### Backend
- FastAPI
- Pydantic

### ML / AI
- scikit-learn (planned)
- sentence-transformers (planned)
- OpenAI API (optional)

### DevOps
- Docker
- Docker Compose

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/decisionnexus.git
cd decisionnexus 
```

2. Run Frontend
```bash
cd frontend
npm install
npm run dev
```
3. Run Backend

```bash
cd backend
python -m venv venv
source venv/bin/activate  # macOS / Linux
venv\Scripts\activate     # Windows

pip install -r requirements.txt
uvicorn main:app --reload
```
